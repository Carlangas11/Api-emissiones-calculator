## Description

Backend application solution for Zero Company IO

## Installation

```bash
# rebuild node modules
$ yarn install
```

## Running the app
```bash
# start local DB
$ docker-compose up -d

# run development
$ yarn start

# run watch mode
$ yarn start:dev

# run production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
